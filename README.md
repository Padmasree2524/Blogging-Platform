# 📝 Blogging Platform

A simple and responsive Blogging Platform built using HTML, CSS, and JavaScript. This project allows users to create, edit, delete, and manage blog posts. All blog data is stored in the browser using Local Storage, so it remains available even after refreshing the page.

## 🚀 Features

- Create new blog posts
- Edit existing blog posts
- Delete blog posts
- Store data using Local Storage
- Responsive and modern user interface
- User-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 📂 Project Structure

```
Blogging-Platform/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server or any web browser.
4. Start creating and managing your blog posts.
